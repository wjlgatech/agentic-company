## Key UX Principles

### 1. **No Mode Switching**
- Everything happens in the same chat
- No "enter refinement mode" buttons
- No modal dialogs

### 2. **Progressive Disclosure**
- Start simple
- Add complexity only when needed
- Don't overwhelm with questions

### 3. **Smart Defaults**
- Generate draft as soon as possible
- Use reasonable assumptions
- Let user refine if needed

### 4. **Conversational Feel**
- Questions sound natural
